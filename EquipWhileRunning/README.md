# Equip While Running

Allows equip and unequip actions while running in Valheim.

## Features
- Toggle mod on/off in game
- Configurable hotkey
- Client-side behavior for the local player

## Requirements
- BepInExPack Valheim

## Installation
Install with a Thunderstore-compatible mod manager, or place the DLL in:

`BepInEx/plugins/EquipWhileRunning/`
